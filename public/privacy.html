Что такое Mindes

Mindes — приложение для управления задачами и проектами. Мы уважаем вашу приватность и собираем только те данные, которые необходимы для работы сервиса.

Какие данные мы собираем

  • Аккаунт Google — имя и адрес электронной почты, которые вы предоставляете при входе через Google Sign-In. Используются только для идентификации вашей учётной записи.
  • Содержимое задач — названия проектов, задачи, описания, сроки и прочие данные, которые вы вводите в приложение. Хранятся в вашем персональном разделе базы данных Firebase Firestore.
  • Фоновые изображения — фотографии, которые вы загружаете в качестве фона, хранятся в Firebase Storage.
  • Локальные настройки — тема оформления и состояние обучения сохраняются только на вашем устройстве (localStorage).

Как мы используем данные

  • Для отображения ваших проектов и задач на всех устройствах.
  • Для синхронизации данных между устройствами в реальном времени.
  • Для отправки локальных уведомлений-напоминаний о задачах (только если вы это разрешили).

Мы не продаём ваши данные третьим лицам, не используем их для рекламы и не передаём без вашего явного согласия.

Сторонние сервисы

Приложение использует инфраструктуру Google:

  • Firebase Authentication — вход через Google.
  • Firebase Firestore — облачное хранение задач.
  • Firebase Storage — хранение фоновых изображений.

На эти сервисы распространяется Политика конфиденциальности Google.

Хранение и безопасность

Данные хранятся на серверах Google (Firebase) с разграничением доступа: каждый пользователь может читать и изменять только свои данные. Передача данных защищена протоколом HTTPS.

Ваши права

  • Удаление данных — вы можете удалить все свои проекты внутри приложения. Для полного удаления аккаунта напишите нам.
  • Экспорт — данные хранятся в вашем аккаунте Firebase и доступны для экспорта по запросу.

Дети

Приложение не предназначено для детей младше 13 лет. Мы осознанно не собираем данные несовершеннолетних.

Изменения политики

При существенных изменениях мы обновим дату вверху страницы. Продолжение использования приложения после изменений означает согласие с обновлённой политикой.
